# BookMyShoot

A marketplace web app where clients in Sri Lanka can book photographers, videographers and editors, and where those pros can receive jobs.

## Project Structure

```
book-my-shoot/
├── frontend/          # React frontend application
├── backend/           # Node.js/Express backend API
├── database/          # MySQL schema and migrations
├── docs/              # Documentation and API specs
└── docker/            # Docker configuration
```
