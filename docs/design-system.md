# BookMyShoot Design System

## Color Palette

### Primary Palette Options

#### Option 1: Professional Teal Theme
- Primary: `#008080` (Teal)
- Secondary: `#F5F5F5` (Light Gray)
- Accent: `#FF6B35` (Orange)
- Background: `#FFFFFF` (White)
- Text: `#333333` (Dark Gray)
- Success: `#4CAF50` (Green)
- Warning: `#FFC107` (Amber)
- Error: `#F44336` (Red)

#### Option 2: Navy & Coral Theme
- Primary: `#0A2F5A` (Navy Blue)
- Secondary: `#F8F9FA` (Light Gray)
- Accent: `#FF6B6B` (Coral)
- Background: `#FFFFFF` (White)
- Text: `#212529` (Dark Gray)
- Success: `#28A745` (Green)
- Warning: `#FFC107` (Amber)
- Error: `#DC3545` (Red)

## Typography

### Font Family
- Primary: Inter (Fallback: system-ui, -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto)
- Monospace: 'SF Mono', Monaco, Inconsolata, 'Courier New'

### Font Sizes
- Heading 1: 2.5rem (40px)
- Heading 2: 2rem (32px)
- Heading 3: 1.5rem (24px)
- Heading 4: 1.25rem (20px)
- Body Large: 1.125rem (18px)
- Body Regular: 1rem (16px)
- Body Small: 0.875rem (14px)
- Caption: 0.75rem (12px)

### Font Weights
- Light: 300
- Regular: 400
- Medium: 500
- Semi Bold: 600
- Bold: 700

## Spacing System
- XS: 0.25rem (4px)
- S: 0.5rem (8px)
- M: 1rem (16px)
- L: 1.5rem (24px)
- XL: 2rem (32px)
- XXL: 3rem (48px)

## Border Radius
- Small: 4px
- Medium: 8px
- Large: 16px
- Circle: 50%

## Shadows
- Light: 0 1px 3px rgba(0,0,0,0.12), 0 1px 2px rgba(0,0,0,0.24)
- Medium: 0 3px 6px rgba(0,0,0,0.16), 0 3px 6px rgba(0,0,0,0.23)
- Heavy: 0 10px 20px rgba(0,0,0,0.19), 0 6px 6px rgba(0,0,0,0.23)

## Components

### Buttons
Primary Button:
- Background: Primary color
- Text: White
- Padding: 0.75rem 1.5rem
- Border Radius: 8px
- Hover: Darken background by 10%

Secondary Button:
- Background: Transparent
- Border: 1px solid Primary color
- Text: Primary color
- Padding: 0.75rem 1.5rem
- Border Radius: 8px
- Hover: Background with 10% opacity of Primary color

### Cards
- Background: White
- Border Radius: 8px
- Shadow: Light
- Padding: 1.5rem

### Inputs
- Height: 44px
- Border: 1px solid #ced4da
- Border Radius: 4px
- Padding: 0.75rem
- Focus: Border color changes to Primary color with glow effect

### Navigation
- Background: White
- Height: 64px
- Shadow: Light bottom shadow
- Link Color: #333333
- Link Hover: Primary color

## Breakpoints
- Small (sm): 640px
- Medium (md): 768px
- Large (lg): 1024px
- Extra Large (xl): 1280px
- 2X Large (2xl): 1536px

## Sri Lanka Specifics
- RTL Support: Not required (LTR languages)
- Date Format: DD/MM/YYYY
- Time Format: 24-hour (HH:mm)
- Currency: LKR (Rs.)
- Number Format: 1,000.00 (comma as thousand separator, dot as decimal)
